# Customer-Analytics-Project
Customer analytics project using a synthetic dataset(from kaggle) to analyze sales trends and customer behaviour in the EU FMCG market.

This project is inspired by overall market trends for hygiene and health categories in Western Europe:
 - Consumer downtrading
 - Weak seasonal demand
 - Rationalized wellness spending
 - Market maturity

# Business Problem
Losing volume, threats from private-labels, competitor's promotion, and lagging sales data visibility are limiting factors for timely decision-making. In this regards, it is meaningful to make consumer-level insight by understanding where share is being lost, to whom, and what behaviour change would recover it.
